# Hammerspoon Integration

Status: temporarily disabled.

The supported surface for `pomodoro_tracker` is the full Phoenix LiveView app:

- development: `mix phx.server`
- production: `./bin/serve`

The repo still keeps `hammerspoon/pomo.lua` as a stub so an existing `~/.hammerspoon` setup can be reloaded safely without recreating the menubar or floating panel.
